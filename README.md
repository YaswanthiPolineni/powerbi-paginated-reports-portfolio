## Overview
This paginated report was developed using **Power BI Report Builder** and is based on a **Power BI Semantic Model** as the data source.  
The report provides a **quarterly sales summary** with a focus on customer-level and state-level performance.

## Report Details
The report displays the following metrics:
- **Total Quantity**
- **Average Price per Unit**
- **Total Sales**

These metrics are grouped by:
- **State**
- **Customer Name**
- **Quarter**

## Key Features
- Uses a **Power BI Semantic Model** as the dataset
- **DAX-based query** with `SUMMARIZECOLUMNS`
- **Region parameter** to dynamically filter data based on customer region
- Supports multi-value parameter selection
- Optimized for **print-ready, pixel-perfect layout**

## Report Layout
### Header
- Report Title
- Company / Branding Image

### Footer
- Report Execution Time
- Page Number (Page X of Y)
